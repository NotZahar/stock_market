### Биржа с клиент-серверной архитектурой

##### Dependencies:
- Boost v1.82
- SQLiteCpp v3.3 [ref](https://github.com/SRombauts/SQLiteCpp)
- CMake v3.28
- Doxygen v1.9

##### Build:
```
mkdir build && cd build
cmake -DCMAKE_BUILD_TYPE=Release ..
cmake --build .
```
